# Release v0.7.0

**Date:** 2026-01-31
**Feature:** F034 - A+ Quality Initiative

> **Note:** Consolidated release notes (0.4.0→0.7.0) — [v0.7.0-release-notes.md](v0.7.0-release-notes.md)

## Overview

SDP достиг уровня A+ по качеству кода, архитектуре и документации. Coverage вырос до 85%, внедрена Clean Architecture, добавлена команда `sdp status`, оптимизированы skills и структура репозитория.

---

## New Features

### F034: A+ Quality Initiative

**Что нового:**
- **Coverage 68% → 85.28%** — превышение целевых 80%
- **Clean Architecture** — domain layer извлечён, нет beads→core нарушений
- **`sdp status`** — статус проекта и guard
- **Skill discovery** — `@help` / `sdp skills list`
- **Skills оптимизированы** — ~64% сокращение объёма
- **Legacy code удалён** — ~600 LOC
- **Репозиторий реорганизован** — root очищен, docs структурированы
- **Lint clean** — F401, F821, E501 исправлены

**Использование:**

```bash
# Статус проекта и guard
sdp status

# Список skills
sdp skills list
```
