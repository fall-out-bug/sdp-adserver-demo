## Overview

F020 — Git hooks extracted to Python, project-agnostic configuration. Hooks now run on SDP itself (dogfooding), auto-detect project root, support `quality-gate.toml`.

---

## New Features

### F020: Fast Feedback (Hooks Extraction & Project-Agnostic)

**Что нового:**
- Shell hooks extracted to Python (`src/sdp/hooks/`) — pre-commit, pre-push, post-build, pre-deploy
- Shell scripts → thin wrappers (<20 lines each)
- `find_project_root()` — markers: `.sdp-root`, `docs/workstreams/`, pyproject.toml
- `find_workstream_dir()` — config: quality-gate.toml, SDP_WORKSTREAM_DIR env
- Hooks work on SDP repository (zero external dependencies)
- Coverage 92%, mypy --strict

**Использование:**

```bash
# Run pre-commit checks (auto-detects project root)
python -m sdp.hooks.pre_commit

# Configuration (quality-gate.toml)
[workstreams]
dir = "docs/workstreams"
```
