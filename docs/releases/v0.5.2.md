## New Features

### pip-audit Security Scanning

- pip-audit в dev-зависимостях
- Сканирование на каждом PR/push в GitHub Actions
- Блокировка merge при обнаружении уязвимостей
- PR-комментарии с деталями (CVE, severity, fix versions)
- Dependabot: еженедельные PR для Python и GitHub Actions
- SECURITY.md и документация по политике зависимостей

**Использование:**

```bash
# Локальное сканирование
poetry run pip-audit

# JSON-отчёт
poetry run pip-audit --format json --desc -o audit-report.json
```
